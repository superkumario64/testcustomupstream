Step-by-Step

Use .list-stepped to make an ordered list look more like a step by step tutorial. Nesting is allowed but use with caution.

  1. Create an ordered list.
  2. Add the class .list-stepped to the <ul>.
  3. Write your steps.
    1. If you want to nest steps now is the time.
  4. Watch your pupils shine.
